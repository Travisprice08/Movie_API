My Movie API

To build the server-side component of a “movies” web application.
The web application will provide users with access to information about different movies, directors, and genres.
Users will be able to sign up, update their personal information,
and create a list of their favorite movies.

API Endpoints

Business Logic HTTP Method URL Query Parameter Request Body Response Body
Return a list of all movies GET /movies none none A JSON object holding data about all movies
Return data on a single movie title GET /movies/[title] /movies/:title none A JSON object holding data about a specific movie

{'Title': 'My Cousin Vinny'
'Starring': 'Joe Pesci'
'Director': 'Jonathan Lynn'
'Genre':'Comedy'}
Return movies based on genre GET /movies/genre/[name] /movies/genre/:name none A JSON object holding data about a movie genre
{'Name': 'Thriller',
'Description': 'Thriller films are films that created a heightened sense of suspense, anticipation, excitement, and surprise. It also tends to have overlapping subgenres.'
}
Return data about the Director (bio and filmography) GET /movies/directors/[name] /movies/directors/:name none A JSON object holding data about the director.

{ 'Name': 'Ridley Scott'
'Biography': 'Directors biography'
'Filmography': 'Directors Filmography'}

Removing movies from favorites DELETE /users/[Username]/movies/[movieID] /users/[Username]/movies/:movieID none A JSON object holding data about updated favorites list
Deleting a user account DELETE /users/:name none none A response that the user account has been deleted
